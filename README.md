Natural Selection
=================

An attempt at getting this mod to compile for Windows and Linux.

See http://unknownworlds.com/ns/ for information about the mod itself.

Dependencies
------------

libpng 1.0.66

STLPort 5.2.1

zlib 1.2.8


Required Linux packages
-----------------------

gcc-multilib

lib32z1-dev

libcurl-dev:i386
